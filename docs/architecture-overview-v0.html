dev-control-tower Architecture

AI 研发控制台 - 基于 Monorepo 的多 Agent 协作流水线管理系统

前端技术栈

  • • React 19 + TypeScript
  • • Vite 构建工具
  • • Tailwind CSS 样式
  • • Zustand 状态管理
  • • Lucide React 图标库

后端技术栈

  • • Express + TypeScript
  • • SQLite (better-sqlite3)
  • • WebSocket 实时通信
  • • Pipeline Scheduler
  • • OpenClaw/Claude 集成

核心业务模块

  • • Agent 管理 - 多 Agent 协作
  • • Pipeline Templates - 流水线模板
  • • Pipeline Instances - 流水线实例
  • • Tasks - 任务调度与执行
  • • Components - 流水线组件库
